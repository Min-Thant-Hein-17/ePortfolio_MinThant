# Min Thant Hein — Personal ePortfolio

A clean, responsive single-page ePortfolio built with vanilla HTML, CSS, and JavaScript. Designed to showcase my academic background, technical skills, certifications, and real-world AI & Data Science projects.

🌐 **Live Site:** [minthanthein.github.io/eportfolio](https://minthanthein.github.io/eportfolio)

---

## Overview

This portfolio serves as my professional digital identity — a central place where recruiters, collaborators, and the tech community can learn about my work, skills, and journey as a Data Science & AI student and engineer.

---

## Sections

- **About** — Background, education, and what drives me
- **Skills** — Technical skills and tools I work with daily
- **Certifications** — Industry certifications from AWS, Google, Microsoft, and more
- **Projects** — Hands-on AI, Data, and Web projects with GitHub links
- **Blog/Articles** — Writings on AI, data engineering, and tech trends
- **Contact** — Get in touch via form or social links

---

## Tech Stack

- HTML5 — semantic, accessible markup
- CSS3 — custom dark theme, responsive layout, smooth animations
- JavaScript (vanilla) — typewriter effect, scroll animations, filter tabs, mobile menu

---

## Features

- Dark-themed UI with teal accent color scheme
- Animated typewriter hero subtitle
- Scroll-triggered section reveal animations
- Project filter tabs (All / AI-ML / Data / Web)
- Fully responsive across mobile, tablet, and desktop
- Downloadable CV button

## Getting Started

```bash
git clone https://github.com/YOUR-USERNAME/eportfolio.git
cd eportfolio
open index.html
```

No dependencies or build tools required — just open `index.html` in any browser.

---

## Deployment

This site is hosted via **GitHub Pages**. Any push to the `main` branch automatically updates the live site.

To enable GitHub Pages on your fork:
1. Go to **Settings → Pages**
2. Set source to `main` branch, `/ (root)`
3. Save — your site will be live in seconds
